## HIGH (integritas Economic Memory â€” thesis #2)

### H-A PaymentRouter self-settlement wash Tier-1 tanpa modal keluar
`src/PaymentRouter.sol:28-49`. `pay()` tak cek `to != msg.sender` maupun kesamaan
controller. Operator register 2 agent (controller = EOA sama), `pay(A,B,X)` â†’ `to =
controllerOf(B) = msg.sender`, X balik ke pengirim di tx yang sama. Net = gas. Cetak
Tier-1 Settlement Success utk A DAN B dengan counterparty linkage NYATA (lebih
meyakinkan dari vault). **Fix:** tolak `to == msg.sender` + tolak
`controllerOf(toAgentId) == controllerOf(fromAgentId)`.

### H-B TaskEscrow self-dealing mint entry Tier-2 (premium) biaya ~gas
`src/TaskEscrow.sol` confirm/createTask/acceptTask. Satu-satunya proteksi kesamaan =
`clientId == workerId`. Operator kontrol client-agent + worker-agent BERBEDA â†’ lifecycle
createTaskâ†’acceptTaskâ†’confirm net 0 (bond dua sisi balik ke operator sama), cetak Tier-2
Agreement Success (sinyal trust tertinggi kedua). "Cost-of-forgery" escrow patah.
**Fix:** larang task di mana `controllerOf(clientId) == controllerOf(workerId)` (cek di
createTask & acceptTask), atau bobot-nol entry bila satu controller.
